Routing to the Google APIs
  1. First access is Google Meta DataServer : 169.254.169.254
  2. restricted.googleapis.com (vpc enabled apis) : 199.36.154.4/30 = (.4 .5 .6 .7)
  3. private.googleapis.com (don't go over public inet) : 199.36.153.8/30 = (.8 .9 .10 .11)
  4. *.gcr.io maps to restricted.googleapis.com
  5. compute.googleapis.com : is not yet in the private set will be soon in beta
  6. Consider the other domains *.googlecloud.com, *.google.com, *.google-analytics.com
